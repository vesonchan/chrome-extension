使用这个页面来测试您的 Vimium Clone 插件的所有功能

📋 使用说明

1. 确保已安装并启用 Vimium Clone 插件

2. 在此页面上测试下面列出的所有快捷键

3. 如果某些功能不工作，请检查浏览器控制台（F12）查看错误信息

⚠️ 故障排除

  • x 和 t 不工作：检查浏览器控制台是否有错误
  • 长按滚动不顺滑：可能需要调整滚动参数
  • 链接提示不显示：确保页面完全加载
  • 标签页操作失败：检查插件权限设置
